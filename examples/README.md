# Examples for Stock Analysis Tool

This folder contains examples demonstrating how to use the Stock Analysis Tool package.

## Example Files

- `basic_analysis.py`: Basic stock analysis with technical indicators and visualization
- `transaction_tracking.py`: How to record, analyze and track investment transactions
- `portfolio_analysis.py`: Complete example combining analysis and transaction tracking

## Running the Examples

```bash
# Make sure you've installed the package first
pip install -e .

# Run basic analysis example
python examples/basic_analysis.py

# Run transaction tracking example
python examples/transaction_tracking.py

# Run complete portfolio analysis example
python examples/portfolio_analysis.py
```

## Menggunakan Contoh Sebagai Referensi

File-file contoh ini dibuat untuk memberikan referensi penggunaan API dari Stock Analysis Tool. Silakan sesuaikan dengan kebutuhan Anda atau gunakan sebagai referensi untuk membuat analisis dan strategi Anda sendiri.
